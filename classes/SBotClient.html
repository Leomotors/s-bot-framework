    Parameters

    • Optional options: SBotOptions

      Options has 5 fields

      • token: Bot Token, Defaults to process.env.DISCORD_TOKEN

      • activityRefreshInterval (Optional): Default 600 seconds

      • logLocation: Location for logging, Default is deactivated

      • ignorePrivacy: If set to true, bot will not deafen when joining voice channel, PS: this bot does not listen to voice chat

      • logIncomingMessage: If set to true, will report every incoming message just like older version of Salim Bot
